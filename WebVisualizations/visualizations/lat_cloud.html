Cloudiness

This graph plots the latitude versus the cloud cover as a percentage. We have a some patterning happening around specific percentages, note the almost linear pattern along the 0, 5, 20, 40, 75, 90 percent mark. From this, we can conclude that measurments are rounded before being logged, which skews the data. Additionally this pattern occurs on latitudes greater in the southern hemisphere which, upon further investigation, indicates that the dataset has more souther hemisphere cities.
